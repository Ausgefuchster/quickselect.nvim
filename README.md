# 🏁 quickselect.nvim

Quickselect is inspired by [wezterm](https://github.com/wez/wezterm).
It allows you to quickly jump to patterns in the buffer.

## 📦 Installation

With [lazy.nvim](https://github.com/folke/lazy.nvim)
```lua
{
    "ausgefuchster/quickselect.nvim",
    event = "VeryLazy",
    opts = {
        -- soon; patterns = {}
    },
}
```
